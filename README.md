## Wodociągi
Program powstał na potrzeby pracy inżynierskiej „GIS w wodociągach. Aplikacja do zarządzania wodomierzami”, zrealizowanej na Akademii Górniczo-Hutniczej w Krakowie pod opieką dr inż. Michała Lupy.

## Technologie
* ASP.NET Core 2.1
* OpenLayers
* PostgreSQL, PostGIS
